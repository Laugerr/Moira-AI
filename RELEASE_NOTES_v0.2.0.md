# 🎮 Moira-AI v0.2.0 Release Notes

`v0.2.0` is the first truly playable version of Moira-AI.

## ✨ Highlights
- Expanded scenario library across multiple life domains
- Scenario domains such as work, education, health, housing, money, family, social, and growth
- Better age progression with more appropriate events across the life arc
- State-aware scenario selection based on life stage and weakest needs
- Persistent `life_flags` system for connected consequence chains
- Follow-up scenarios across work, risk, money, relationships, education, and housing
- First real balancing pass across the wider event pool
- Clearer UI feedback for needs, history, and run state

## 🧩 What This Release Proves
- Moira-AI is now meaningfully replayable
- Different runs can produce different life paths
- Choices can echo forward instead of disappearing after one turn
- The prototype is strong enough to support deeper systems in `v0.3.0`

## 🔜 Next Focus
`v0.3.0` will turn the playable life loop into a more systems-heavy simulator by introducing:
- player-driven actions
- deeper progression systems
- stronger long-term identity and consequences
