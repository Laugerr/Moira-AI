# 🚀 Moira-AI v0.1.0 Release Notes

`v0.1.0` is the first usable prototype of Moira-AI.

## ✨ Highlights
- Working Flask-based life simulation loop
- Start flow, choice flow, and ending flow
- Needs-based stat model:
  - money
  - health
  - energy
  - happiness
  - social
- Expanded scenario library with JSON loading and fallback scenarios
- Cleaner prototype UI with less clutter
- History timeline and profile snapshot updates
- Improved ending titles and life-summary style feedback

## 🧩 What This Release Proves
- The core concept works
- A full life run can be played from start to ending
- The current prototype is solid enough to build `v0.2.0` on top of

## 🔜 Next Focus
`v0.2.0` will turn the prototype into the first truly playable version by improving:
- scenario depth
- replayability
- balancing
- UI clarity
- overall feel
